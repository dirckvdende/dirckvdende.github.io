A small puzzle web app. In each puzzle you are presented with a set of chess pieces and a board that the pieces can be placed on. The goal is to place the pieces in such a way that none can attack each other. There are many levels available! Some are quite difficult :)

{% include center-button.html href="https://dirck.dev/no-mate/" text="No Mate web app" icon="link" %}

The web app runs entirely in the user's browser. It was built using Vue.js and published on GitHub pages. The source code of the project can be found on [GitHub](https://github.com/dirckvdende/no-mate). If you have a suggestion for a puzzle, please let me known here too by making an issue and/or pull request.
